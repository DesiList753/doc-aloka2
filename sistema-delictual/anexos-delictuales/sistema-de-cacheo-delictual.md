# Sistema de Cacheo Delictual

## Sistema de Cacheo Delictual

{% hint style="info" %}
Este criterio ya fue integrado en [Actos Delictuales](../actos-delictuales.md).
{% endhint %}

### Qué regula este anexo

El cacheo delictual dentro de escenas criminales activas.

### Cuándo es válido

* existe acto delictual activo y justificado (asalto, secuestro o guerra territorial).
* existe control efectivo de escena.
* ocurre fuera de zona segura.

### No válido

* cachear en pleno fuego cruzado.
* sacar objetos infinitos.
* asumir hallazgos instantáneos sin rol.
