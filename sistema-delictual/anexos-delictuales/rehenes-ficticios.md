# Rehenes Ficticios

## Rehenes Ficticios

{% hint style="info" %}
Este criterio ya fue integrado en [Actos Delictuales](../actos-delictuales.md).
{% endhint %}

### Regla base

Los rehenes ficticios solo existen si el sistema o staff los valida.

### Restricción

No pueden usarse para inflar exigencias o forzar ventajas absurdas.

### Prioridad

Si el rehén no aporta rol creíble, no se considera válido.
