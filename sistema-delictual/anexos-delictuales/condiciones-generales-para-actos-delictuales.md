# Condiciones Generales para Actos Delictuales

## Condiciones Generales para Actos Delictuales

{% hint style="info" %}
Este criterio ya fue integrado en [Actos Delictuales](../actos-delictuales.md).
{% endhint %}

### Siempre debes respetar

* proporcionalidad.
* preparación.
* continuidad.
* fair play.
* capacidad real del grupo.

### No puedes

* encadenar delitos sin descanso para abusar del sistema,
* usar personajes secundarios como herramientas desechables,
* romper el ambiente del servidor por beneficio rápido.

El acto delictual válido deja historia, no solo loot.
