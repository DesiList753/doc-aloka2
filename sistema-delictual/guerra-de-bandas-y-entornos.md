# Guerra de Bandas y Entornos

## Guerra de Bandas y Entornos

Esta guía operativa define cómo ejecutar conflictos territoriales entre bandas y cómo reportarlos correctamente con /entorno.

Aplica solo para **bandas oficiales**.

{% hint style="info" %}
Las reglas base del sistema de grafitis se mantienen en [Normativa Graffitis](normativa-graffitis.md).
{% endhint %}

### Cuándo usar esta guía

Úsala cuando exista:

* disputa territorial activa entre bandas.
* escalamiento de conflicto en zona de grafitis.
* necesidad de informar inicio, cambios o cierre del conflicto por /entorno.

### Objetivos válidos para iniciar guerra territorial

Además del control de territorio, también se puede iniciar conflicto cuando exista un motivo de peso relacionado con:

* robo o recuperación de suministros de interés criminal (armamento, munición, chalecos o recursos de operación).
* obtención de información útil para el conflicto (movimientos, contactos o puntos de operación).
* identificación de integrantes rivales mediante datos de rol, como información de DNI obtenida de forma contextual y justificada.

{% hint style="warning" %}
Esto no habilita el robo de documentos físicos ni contradice las reglas generales de [Compendio Delictual](compendio-delictual.md).
{% endhint %}

### Guía de ejecución (guerra de bandas territoriales)

La guerra de bandas debe tener contexto, motivo y objetivo territorial claro.

No es válido iniciar conflicto solo para forzar DM o generar caos sin continuidad de rol.

#### Flujo de ejecución (inicio, desarrollo y cierre)

1. definir el motivo del conflicto y el territorio en disputa.
2. enviar entorno de inicio antes de la primera acción hostil.
3. mantener el conflicto dentro del área declarada.
4. si cambia la zona o escala la escena, enviar entorno de actualización.
5. al terminar, enviar entorno de cierre para dejar constancia.

#### Límites operativos (durante el conflicto)

* Solo pueden participar miembros oficiales presentes en la escena.
* No se permiten refuerzos externos fuera de las reglas generales de organizaciones.
* No se puede extender la guerra de forma artificial para farmear enfrentamientos.
* El cacheo en ciudad es válido si es consecuencia directa del conflicto, existe control de escena y ocurre fuera de zona segura.
* Si interviene **Carabineros de Alokados**, el conflicto pasa a contexto policial activo y se debe ajustar el rol.

### Variaciones de /entorno (para guerra de bandas)

Todo conflicto territorial entre bandas debe avisarse con /entorno claro y coherente.

#### Variación 1 (inicio de conflicto territorial)

Úsalo cuando comienza la disputa en una zona concreta.

Ejemplo:

* /entorno inicio de conflicto territorial entre bandas en [zona], grupo armado en el lugar, posible enfrentamiento en curso.

Puedes añadir el motivo principal del conflicto cuando corresponda:

* /entorno inicio de conflicto territorial entre bandas en [zona] por disputa de suministros e información de identidad.

#### Variación 2 (escalamiento o desplazamiento)

Úsalo cuando llegan más participantes o la escena se mueve de sector.

Ejemplo:

* /entorno escala conflicto territorial entre bandas hacia [zona], aumenta cantidad de involucrados y riesgo para civiles.

#### Variación 3 (repliegue o término parcial)

Úsalo cuando una banda se retira y el conflicto baja de intensidad.

Ejemplo:

* /entorno repliegue de banda desde [zona], conflicto territorial en descenso, mantener precaución en el sector.

#### Variación 4 (cierre total del conflicto)

Úsalo cuando la guerra termina en ese punto y no seguirá de inmediato.

Ejemplo:

* /entorno finaliza conflicto territorial entre bandas en [zona], sin continuidad inmediata.
