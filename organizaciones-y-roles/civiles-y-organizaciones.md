# Civiles y Organizaciones

## Civiles y Organizaciones

### Civiles

Todo civil debe tener objetivos coherentes.

No puede vivir únicamente para delinquir o romper escenas.

### Organizaciones

Toda organización debe tener identidad, estructura y límites claros.

### Prohibido

* crear grupos solo para PvP.
* reciclar nombres para evadir historial.
* mezclar facciones incompatibles sin transición de rol.
