# Uso de Sniper

## Uso de Sniper

### Regla base

El sniper requiere contexto fuerte y ejecución limpia.

No es un arma para abrir rol de forma gratuita.

### Condiciones

* Debe existir un conflicto previo o causa sólida.
* El tirador debe tener posición lógica.
* La escena debe permitir identificación y respuesta.

### Restricciones

* No se usa para matar por sorpresa sin rol previo.
* No se usa en zonas protegidas si la situación lo impide.
* No se usa para arruinar negociaciones sin motivo real.

### Consecuencia

Cada disparo de precisión debe tener impacto narrativo.

Si lo usas, asumes escalada total del conflicto.
