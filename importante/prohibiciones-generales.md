# 🚫 Prohibiciones Generales

## Prohibiciones Generales

### Queda prohibido

* cheats, macros o software externo.
* bug abuse.
* stream sniping.
* compra o venta externa de bienes del servidor.
* suplantación de staff.
* falsificación de pruebas.
* compartir cuentas.

### También se prohíbe

* trolear escenas administrativas.
* evadir consecuencias mediante desconexión.
* usar personajes desechables para delinquir.
* entrar solo a romper rol.

{% hint style="danger" %}
Cualquier uso de herramientas externas para obtener ventaja implica expulsión casi inmediata.
{% endhint %}
