# Jerarquía SAMU

## Jerarquía SAMU

### Estructura base

La cadena interna de SAMU se organiza así:

1. **Jefatura SAMU**
2. **Subdirector SAMU**
3. **Médico Cirujano**
4. **Jefe de Enfermeros**
5. **Psicóloga**
6. **Enfermero o Enfermera**
7. **Auxiliar de Enfermería**
8. **Alumno en Práctica**

### Criterio operativo

Cada rango debe respetar la cadena de mando.

Las decisiones críticas deben seguir criterio médico, jerarquía y contexto de escena.

### Responsabilidad

Tener rango no otorga privilegios fuera de normativa.

Otorga mayor responsabilidad institucional y operativa.
